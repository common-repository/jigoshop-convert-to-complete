== Description ==

Adds the ability to convert bulk orders to complete in Jigoshop

== Installation ==

1. Unzip the archive on your computer  
2. Upload `jigoshop-convert-to-complete` directory to the `/wp-content/plugins/` directory
3. Activate the plugin through the 'Plugins' menu in WordPress
